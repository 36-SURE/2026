Pittsburgh-based Resources

CMU Sponsored ID

A CMU sponsored ID is an optional, additional expense for students that can provide the following:

  • General building access

  • Fitness center access (GroupX classes not included)

  • Library access & borrowing of books

  • CMU campus shuttles & escort services

    • Note that Pittsburgh Bus Transit is not included

Pittsburgh Bus/Transit Pass

If you would like to use public transit to explore what Pittsburgh has to offer, please see the below links for more information:

  • Current bus fares

  • Getting a bus/fare pass

  • Personally, we recommend Google Maps (for planning routes) and the Transit App (for real-time tracking) to coordinate bus plans.

  • UPMC Presbyterian Hospital

    • 200 Lothrop St, Pittsburgh, PA 15213

    • 412-755-2307

  • Allegheny Health Network Emergency Care Brentwood

    • 3290 Saw Mill Run Blvd, Brentwood, PA 15227

  • Resolve Crisis Services

  • 333 North Braddock Ave, Pittsburgh, PA 15213

  • 1-888-796-8226

  • Allies for Health and Well-Being

If you find yourself struggling with having enough food to get through your time here, please contact the following resource for assistance:

  • Greater Pittsburgh Community Food Bank 412-460-3663 ext. 655
